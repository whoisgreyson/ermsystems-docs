It is important to define role permissions for your team to ensure efficient internal operations. With the Basic Settings, you are able to define permissions and your preferred prefix for the server.

## Basic Settings Configuration

1. Run the `/config` command.
2. Enable your staff roles with the first menu, admin roles with the second menu, and management roles with the third menu.
3. Select your server's preferred prefix for text-based commands. At this time, you can only select from the following prefix options: `!` `>` `?` `:` `-` 

<Warning>
  This module is a core feature of ERM. It is always enabled and cannot be disabled.
</Warning>
